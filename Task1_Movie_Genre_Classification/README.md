# Task 1 - Movie Genre Classification (Machine Learning)

## Description
This project builds a Machine Learning model that predicts a movie's genre based on its plot description using Natural Language Processing techniques.

## Features
- Loads and processes movie dataset  
- Extracts genre and description from raw data  
- Cleans and vectorizes text using TF-IDF  
- Trains a Linear Support Vector Machine (SVM) classifier  
- Evaluates model performance with accuracy and classification report  

## How to Run
1. Open terminal  
2. Navigate to the project folder:  
   cd Task1_Movie_Genre_Classification  
3. Run the program:  
   python movie_genre_classifier.py
